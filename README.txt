# Game Instruction:

**LUẬT CHƠI:**
    
    Người chơi sẽ điều khiển chú rắn ăn các mục tiêu trên màn hình.
    
    - Nhấn UP để điều khiển rắn di chuyển lên
    - Nhấn DOWN để điều khiển rắn di chuyển xuống
    - Nhấn RIGHT để điều khiển rắn di chuyển sang phải
    - Nhấn LEFT để điều khiển rắn di chuyển sang trái
